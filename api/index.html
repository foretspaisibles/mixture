The Mixture Mixins Library

Mixture provides standard mixins allowing to quickly prepare consistant programming interfaces.


Mixture_Applicative
Applicative mixin.
Mixture_Compare
Comparable mixin.
Mixture_Format
Formatted mixin.
Mixture_Parse
Parsable mixin.
Mixture_Monad
Monad mixin.
Mixture_FoldRight
Right foldable mixin.
Mixture_FoldAssoc
Foldable associative mixin.
Mixture_NumericalFunction
Numerical function mixin.
