Walking through the values of the function

val fold_step : t ->
       float -> (float -> float -> 'a -> 'a) -> 'a -> 'a
fold_step f step loop acc compute loop x (f x) acc for all x in the range of f with the given step, accumulating state in acc.

If step < 0.0 the range is explored backwards.
Raises Invalid_argument if step = 0.0.

val iter_step : t ->
       float -> (float -> float -> unit) -> unit
iter_step f step loop compute loop x (f x) for all x in the range of f with the given step.
Raises Invalid_argument if step <= 0.0.
val fold_ticks : t ->
       int -> (float -> float -> 'a -> 'a) -> 'a -> 'a
fold_ticks f ticks loop acc compute loop x (f x) acc for n evenly spaced values of x in the range of f, accumulating state in acc.
Raises Invalid_argument if n is smaller than 0.
val iter_ticks : t ->
       int -> (float -> float -> unit) -> unit
iter_ticks f ticks loop compute loop x (f x) for n evenly spaced values of x in the range of f.

if n < 0, the range is explored backwards.
Raises Invalid_argument if n = 0.
